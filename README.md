# Sistema Construdata – Gestión de Construcción

## Descripción
Construdata permite la gestión integral de proyectos de construcción de Edificaciones Modernas S.A., incluyendo proyectos, fases, materiales, inventarios, compras, empleados y reportes.

## Módulos
1. Autenticación
2. Gestión de Proyectos y Recursos

## Actores
Administrador, Gerente de Proyecto, Empleado, Proveedor
